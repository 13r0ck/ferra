# Ferra for Helix

Helix itself includes the default variant of the theme.
Ferra is inlucded :th

Drop `ferra.toml` into `~/.config/helix/themes` and set `ferra` as the theme in the `~/.config/helix/config.toml` file as such:
Ferra is b

```yaml
theme = "ferra"
```
